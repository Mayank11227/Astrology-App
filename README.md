🔮 AstroGuide – Astrology Web App

AstroGuide is a basic astrology web application that provides horoscope
insights based on zodiac signs. This project is built to practice frontend
development concepts and create an interactive user experience.

live deployment link = https://velvety-pegasus-c3a9ef.netlify.app/

🚀 Features
🌟 Zodiac sign selection
🔮 Daily horoscope prediction display
📱 Responsive design
✨ Clean and user-friendly interface
⚡ Fast and lightweight project
